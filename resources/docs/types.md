## Types

### int
````java
int someInt = 10_000_000;
````

ranges: +/- **2.1B**


### String
Uninitialized has a default value **null**

### double
````java
double someDouble = 32.16;

double someDouble = 32.16d;

double someDouble = 32.16D;

double someDouble = 2.1E12;

double someDouble = 75; //Becomes 75.0
        
double someDouble = .3;

double someDouble = 4.0f;
````
